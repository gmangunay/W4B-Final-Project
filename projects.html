Projects

Robotics Project – TI-RSLK MAX Maze Solver

Developed an obstacle avoiding algorithm on the Texas Instruments Robotics
Learning Kit. There were three GP2Y0A21YK0F Infrared sensors untilized
for the obstacle avoidance. Applied different duty cycles for each PWM motor
depending on which sensors were triggered. All of this was done inside the Code
Composer Studio IDE as it was neccessary to access the on-board chips. Prior to
the completion of the project, I also worked on many path-planning algorithms
such as BFS, DFS, Dijkstra, and A* on MATLAB.

Data Acquisition Project – Non-Contact Temperature Gun

Converted a Nerf blaster into a temperature reading non-contact device. The
component responsible for reading the target's temperature is the MLX90614
Infrared sensor. The readings become inaccurate if the sensor are outside of
the optimal range. To combat this, I implemented a HC-SR04 Ultrasonic sensor
to calculate the distance from the target. An LCD screen displays target
temperatures along with distance if the user is too far away. Features such as
LED indicators and a laser diode are also installed for target acquisition and usability.
All of this is powered through an Arduino Nano and wasdeveloped on the Arduino IDE.

Senior Project - LED Electronic Ariel Flare (L.E.A.F)

Designed an autonomous drone that when deployed simulates an advanced becaon
of distress by emitting morse code. Created our own dedicated flight controller module
through an Arduino Nano microcontroller as our drone would only serve the purpose of
being deployed during an emergency (no recreational use). Implemented 18650 Lithium-ion
batteries to power the 4 brushless motors installed. The Adafruit 9-dof Aboslute Orientation
IMU Fusion Breakout was another key component in this project as it allowed us to calculate
the roll, pitch, and yaw of the drone. The whole frame was contructed by 3D printing in order
to save on cost and weight of the project. Project was halted due to the start of the pandemic.
